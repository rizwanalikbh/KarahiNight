ORDER RULES:

Important constraints:




Each invited user can place ONLY ONE order.




A user can choose ONLY ONE pickup slot.




Users cannot split pizzas across multiple slots.




Quantity selection must depend on remaining slot capacity.

Example:

If slot capacity = 3
Already booked = 2

User may only choose:
1 pizza




Maximum quantity allowed must equal:

MIN(
remaining pizzas in selected slot,
remaining pizzas in total event capacity
)




If total event capacity (10 pizzas) is reached:
Disable ordering completely.




If slot capacity is reached:
Disable that slot.




If user already submitted an order:
Show existing order instead of allowing another order.
